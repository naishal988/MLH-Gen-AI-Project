# 🛡️ PhishGuard AI 

An advanced, Zero-Trust Architecture based Phishing Detection Engine built for the **MLH Gen AI Hackathon**.

### 🌟 Live Demo
#link

### 🚀 Features
* **Hyper-Paranoid Threat Hunting:** Detects root domain impersonation and typosquatting.
* **Government Spoofing Detection:** Strict rules engine to catch fake `.in` or `.online` government domains.
* **Social Engineering Analysis:** Scans email texts for psychological urgency and scam keywords.
* **Real-time Dashboard:** Tracks total scans, safe links, and phishing attempts dynamically.

### 💻 Tech Stack
* **Backend:** Python, Flask, SQLite (Developed by Naishal)
* **Frontend:** HTML5, CSS3, Vanilla JS, Chart.js (Developed by Sruthika)
* **AI Engine:** Groq API (Llama 3.1 8B Instant) with Temperature 0.0 for deterministic heuristic analysis.
* **Deployment:** Render (Backend) & Netlify (Frontend)

### ⚙️ How it Works
The AI prompt is engineered with a strict Zero-Trust policy. Instead of relying on static blocklists, it performs dynamic DNS structure analysis and flags even a 1% deviation from official root domains.
